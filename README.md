## Meathead

A training app designed to plan, track and optimize your meathead goals.

## 🚀 Getting Started

### 🛠️ Installation

```bash
git clone https://github.com/vieko/meathead.git
cd meathead
pnpm install
```

### 🚀 Usage

```bash
pnpm dev
```

Open [http://localhost:3001](http://localhost:3001) with your browser to see the result.
